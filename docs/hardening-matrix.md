# XCSteward Hardening Matrix

This matrix is the public-alpha safety runbook. A row is complete only when its
command exits successfully and the expected result below is true. "Does not
crash" is not an acceptable expected result.

Run fake-tool rows sequentially from one checkout with
`bash scripts/run-hardening-matrix.sh`; separate concurrent SwiftPM invocations
can contend on shared `.build` and temporary state setup. Use
`bash scripts/run-hardening-matrix.sh --list` to inspect selected rows, or
`bash scripts/run-hardening-matrix.sh --row <row-id>` to run one row.
Each run writes a JSON report to `.build/hardening-matrix/latest-report.json`
by default; pass `--report <path>` to choose a different report location.
Use `--continue-on-failure` when you want one report that inventories every
selected failing row instead of stopping at the first failure.
The runner validates matrix commands and executes only supported
`swift test --filter ...` argv forms; it does not evaluate row text through a
shell. Each selected row is executed through
`scripts/run-swift-test-filter.sh`, so a stale filter that matches zero tests is
a release-gate failure instead of a SwiftPM warning hidden behind exit code 0.
`HardeningMatrixTests` verifies that the runner lists exactly these fake rows
and that row IDs remain unique, so the executable gate cannot silently drift
from this manifest.
Run the live smoke row only on a prepared local Mac with a real iOS Simulator
UDID via `bash scripts/run-hardening-matrix.sh --include-live`.
